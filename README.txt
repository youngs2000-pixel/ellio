  🌍 세계 3차 전쟁  v2.0

================================================================

■ 가장 빠르게 해보기 (인터넷/서버 없이)
  - index.html 파일을 더블클릭하면 바로 게임이 됩니다.
  - 이때는 "AI 내러티브(세계 뉴스·외교 해설)"가 인터넷 대신
    내장된 한국어 문구로 나옵니다. 게임 규칙·재미는 100% 똑같습니다.
  - 즉, 서버를 안 켜도 처음부터 승리/패배까지 전부 플레이됩니다.


■ 게임 방법 (한눈에)
  1) 시작 화면에서 ① 난이도(쉬움/보통/어려움)와
     ② 플레이할 나라(12개 중 하나)를 고릅니다.
  2) 한 턴에 행동 2번(⚡)을 할 수 있어요. 자원(💰)을 써서:
       - 내 나라 강화: 군사/경제/외교/기술
       - 방어 태세 / 국가 재정비 / 평화 회담 / 핵무기 개발
       - (지도에서 다른 나라 클릭 후) 동맹·점령·첩보·경제제재·핵 압박
  3) 모든 나라를 동맹 또는 점령하면 → 세계 통일 승리! 🎉

  ★ v2.0 핵심 재미
   - 능력치는 1~1000까지 자라요(1000에 가까울수록 천천히 오름).
   - 전쟁을 하면 이긴 쪽도 군사·자원·"전쟁 피로도(😣)"가 깎여요.
     → 너무 많이 싸우면 우리도 약해지고, 다른 나라가 노려요.
   - 핵무기(☢️)는 아주 강하지만 위험해요.
     너무 쓰면 "세계 파국 위기"가 와서 패배할 수 있고, 점수도 크게 깎여요.
   - 같은 지역 나라를 모두 내 편으로 만들면 "지역 통합 보너스"!
   - 거의 다 이겼을 때 "세계 저항 연합"이 나타나 마지막 긴장감을 줍니다.
   - 핵을 한 번도 안 쓰고 동맹 위주로 통일하면 "평화적 통합" 특별 엔딩!


■ 숫자(밸런스) 바꾸기 — 아이와 함께 실험!
  - index.html 을 메모장/VS Code로 열면 <script> 맨 위에
    "BAL" 이라는 큰 표가 있어요. 거의 모든 밸런스 숫자가 여기 모여 있습니다.
    예) 시작 자원, 강화 비용, 전쟁 소모율, 핵 조건, 점수 기준 ...
  - 여기 숫자만 바꿔도 난이도와 재미가 확 달라져요.
  - 지도 마커 위치가 어긋나면 COUNTRY_DATA 의 fx, fy(0~1 비율)를 조절하세요.

  ※ 밸런스 자동 시험 (선택, Node.js 필요)
     명령창에서:  node sim.js 40
     → 난이도/국가/전략별 승률·평균턴·점수를 자동으로 뽑아줍니다.
       (숫자를 바꾸고 다시 돌려서 균형을 확인하는 용도)
     참고 균형(똑똑한 플레이 기준, 대한민국):
       쉬움 ~90% · 보통 ~68% · 어려움 ~43%
       순수 외교만/무지성 공격만 = 0% (한 가지 전략만 반복하면 짐)


================================================================
  ■ AI 내러티브를 "진짜 AI(Google Gemini)"로 켜기 — Vercel 배포
================================================================
  서버를 통해 매번 다른 세계 뉴스·외교 메시지·엔딩을 진짜 AI가 써 줍니다.
  (API 키는 절대 게임 파일/브라우저에 넣지 않고 서버에만 둡니다.)

  준비물
   - GitHub 계정 (저장소: youngs2000-pixel/ellio)
   - Vercel 계정 (GitHub로 로그인)
   - Google AI Studio 의 Gemini API 키
       발급: https://aistudio.google.com/apikey

  1) GitHub에 올리기 (이 폴더에서, 명령창)
       git init
       git add .
       git commit -m "세계 3차 전쟁 v2.0"
       git branch -M main
       git remote add origin https://github.com/youngs2000-pixel/ellio.git
       git push -u origin main
     ※ .gitignore 덕분에 .env(비밀키)와 기획 문서/스크린샷은 안 올라갑니다.

  2) Vercel에서 배포
       - vercel.com → Add New → Project → GitHub의 "ellio" 가져오기
       - Application Preset: Other,  Root Directory: ./  (그대로)
       - Environment Variables 에 추가:
            Key   : GEMINI_API_KEY
            Value : (발급받은 Gemini 키)
         (원하면 Key: GEMINI_MODEL, Value: gemini-2.0-flash 도 추가)
       - Deploy 클릭 → 잠시 뒤 https://ellio-....vercel.app 주소가 나옵니다.
       - 그 주소로 들어가면 게임 + 진짜 AI 내러티브가 함께 동작합니다.

  3) 동작 원리 (안심 포인트)
       브라우저(index.html) → /api/ai (Vercel 서버 함수) → Gemini
       - 키는 서버 함수(api/ai.js) 안에서만 사용 → 사용자에게 노출 안 됨.
       - AI가 게임의 수치/승패는 절대 못 바꿔요. "글(문구)"만 만듭니다.
       - 인터넷이 끊기거나 AI가 실패해도, 자동으로 내장 문구로 바뀌어
         게임은 멈추지 않습니다.
       - 시작 화면의 "🤖 AI 내러티브" 스위치로 언제든 끄고 켤 수 있어요.

  비용 메모
   - 호출은 턴당 뉴스 1회, 외교/핵 사용 시, 5턴마다 전략 분석, 엔딩 1회 정도로
     최소화했습니다. Gemini Flash 모델은 매우 저렴하지만,
     아이와 쓰는 용도라면 Google AI Studio에서 사용량을 가끔 확인하세요.
